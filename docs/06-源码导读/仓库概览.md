# 仓库概览

## 概述

Univona 由 6 个核心仓库构成，分别承担共享协议、服务端、客户端与部署环境的职责。

## 仓库列表

| 仓库 | 说明 |
|------|------|
| `univona` | 主仓库：共享类型库 + Protobuf 协议 + 文档 |
| `univona-admin-server` | 官方中枢服务端（Panel + Daemon + 目录/KYC） |
| `univona-daemon` | 社区服务端（单社区消息路由核心） |
| `univona-panel` | 多 Daemon 管理面板 |
| `univona-client` | 跨平台客户端（Flutter + Rust 核心） |
| `univona-docker-dev` | Docker 开发环境 |
