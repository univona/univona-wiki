# 测试指南

## Rust 后端

```bash
# 单元 + 集成测试
cargo test --workspace
```

## Admin Server 性能测试

```bash
# 需要安装 k6
k6 run tests/performance/http-api.js
k6 run -e AUTH_TOKEN=your-jwt tests/performance/websocket.js
```

## Flutter 客户端

```bash
flutter test
flutter analyze
```
