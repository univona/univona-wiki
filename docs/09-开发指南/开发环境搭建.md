# 开发环境搭建

## 概述

Univona 同时包含 Rust 服务端与 Flutter 客户端，建议使用 Docker 开发环境快速起步。

## 依赖要求

- Rust 1.92+
- Node.js 20+
- PostgreSQL 16+
- Redis 7+（可选）
- Flutter 3.x（客户端开发）

## 推荐方式：Docker 开发环境

```bash
cd univona-docker-dev
bash scripts/setup-dev.sh
```

默认启动 PostgreSQL + Redis + pgAdmin（可选）。

## 服务端本地运行

```bash
# Admin Server
cd univona-admin-server
cargo run --release

# Daemon
cd univona-daemon
cargo run --release
```

## 客户端运行

```bash
cd univona-client/flutter
flutter pub get
flutter run
```
