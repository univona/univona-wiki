# 代码风格

## Rust

- 使用 `rustfmt` 统一格式化
- 使用 `clippy` 进行静态检查（禁止 warnings）

```bash
cargo fmt --all
cargo clippy --all-targets --all-features
```

## 前端（React）

- 使用 ESLint + TypeScript 严格模式
- 构建前运行 `npm run build`

## Flutter

- 使用 `dart format` 与 `flutter analyze`

```bash
flutter format .
flutter analyze
```
