# 核心特性

## 概述

Univona 的核心能力覆盖身份、加密、消息系统、媒体与部署全链路，兼顾隐私、安全与可自托管性。

## 功能特性

- **端到端加密**：X3DH + Double Ratchet，前向保密与妥协后安全
- **零注册身份**：Ed25519 密钥对即身份，BIP-39 助记词恢复
- **混合架构**：官方目录 + 社区自建 + P2P 直连
- **多设备同步**：基于游标的增量同步
- **离线消息**：中继存储 + 过期清理
- **媒体处理**：文件上传/缩略图/去重
- **权限控制**：5 级角色 + 18 权限 + 覆盖机制
- **Bot 系统**：可配置权限的机器人
- **推送通知**：FCM / APNs / UnifiedPush
