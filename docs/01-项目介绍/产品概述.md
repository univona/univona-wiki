# 产品概述

## 概述

**Univona** 是一个开源、可自托管的端到端加密即时通讯平台，目标是让用户在隐私、安全与可控性之间拥有更多选择。它将 **官方中枢（目录/KYC）**、**社区自建（Daemon）** 与 **P2P 直连** 三种模式融合在同一套协议与客户端中。

## 核心理念

- **数据主权**：社区可自托管，完全掌控数据
- **隐私优先**：端到端加密默认启用，无需手机号/邮箱
- **开放可扩展**：协议、服务端与客户端均开源，可按需二次开发

## 三种连接模式

1. **Mode A - 完全 P2P**：客户端直连，无服务器依赖
2. **Mode B - 社区 Daemon**：自建社区服务器，类似 Discord 模式
3. **Mode C - Admin Server 联邦**：官方中枢，提供目录与合规能力
