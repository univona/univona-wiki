# 目标用户

## 概述

Univona 面向需要自托管与隐私保护的个人、社区与组织，提供比传统 IM 平台更可控的部署与数据策略。

## 用户画像

### 1. 隐私敏感的个人用户

- 需要端到端加密与匿名身份
- 希望在 P2P 或自建社区中沟通

### 2. 社区与兴趣组织

- 自建服务器，掌控成员与内容
- 需要频道、权限与管理工具

### 3. 企业/机构

- 需要合规能力与审计工具
- 更倾向使用 Admin Server 模式统一管理

### 4. 开发者与研究者

- 需要开放协议与可扩展架构
- 关注可复用的加密与通信组件
